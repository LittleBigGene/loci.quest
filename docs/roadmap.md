# Roadmap

This is a high-level view of where Loci is headed. Dates are not commitments — priorities may shift based on feedback and exam calendar.

## Exam tracks

| Track | Status | Notes |
|---|---|---|
| **6U** — Regulation and Financial Reporting (US) | In development | First release; flagship Domain C wing |
| **6C** — Regulation and Financial Reporting (Canada) | Planned | Same estate model as 6U |

## Product

| Area | Status | Notes |
|---|---|---|
| 2D estate map + domain wings | Shipped (skeleton) | Progress and unlock flow in progress |
| Escape-room task rooms | In development | Hotspots, puzzles, reference readings |
| Mnemonic gallery | In development | Domain D wing |
| Local progress (browser storage) | Shipped | No account required |
| PWA / offline revisits | Planned | Web-first |
| Accounts + progress sync | Planned | **loci.api** + Supabase |
| Purchase flow (Estate $199 / Estate + Noriko $599; add Noriko $399; renew $149/yr) | Planned | Tied to **loci.api** + Stripe; lifetime estate; 1-year Noriko Q&A |
| 3D walkable room interiors | Future | Three.js upgrade; content stays data-driven |

## How to influence the roadmap

Open a **[GitHub Issue](https://github.com/LittleBigGene/loci.quest/issues)** describing the exam track or feature you need and why. Exam-track requests (especially 6C) help prioritize what comes after 6U.
