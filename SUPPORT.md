# Support

This repo is the public home for **Loci** customer support. The app itself is not open source.

## Get help

1. Check **[docs/faq.md](docs/faq.md)** for common questions.
2. Search **[existing issues](https://github.com/LittleBigGene/loci.quest/issues)** — someone may have already reported the same thing.
3. Open a **[new issue](https://github.com/LittleBigGene/loci.quest/issues/new)** if you still need help.

## Reporting bugs

When filing a bug, please include:

- **Browser and OS** (e.g. Safari 18 on macOS, Chrome on Windows)
- **Exam track and URL path** (e.g. CAS-6U, `/cas-6u/d-wing`)
- **What you expected** vs **what happened**
- **Steps to reproduce**, if possible
- **Screenshots** for UI issues

Do not paste exam answers, personal account details, or purchase receipts into public issues.

## Feature requests

Use **[GitHub Issues](https://github.com/LittleBigGene/loci.quest/issues/new)** with a clear description of the problem you are trying to solve. Feature requests for additional exam tracks (e.g. 6C) are welcome — see [docs/roadmap.md](docs/roadmap.md) for what is already planned.

## Billing and access

Purchase and account questions are handled outside this repo. Include your order email when contacting support through the channel provided at checkout (details will be added when the purchase flow launches).

## Security

If you believe you have found a security vulnerability, do **not** open a public issue. Contact the maintainer privately with enough detail to reproduce the issue.
